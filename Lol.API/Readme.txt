PROJEKT INFORMÁCIÓK

Fejlesztési környezet:
- Visual Studio 2022

SZÜKSÉGES LÉPÉSEK A FUTTATÁSHOZ

1. XAMPP elindítása
   - Apache és MySQL szolgáltatások bekapcsolása

2. Adatbázis létrehozása
   - Nyissa meg: http://localhost/phpmyadmin
   - Hozzon létre egy új adatbázist "champs" néven

3. Adatbázis importálása
   - A projektben található:
     Data/Database/*.sql
   - Importálja ezt a fájlt a "champs" adatbázisba

4. Projekt futtatása
   - Nyissa meg a projektet Visual Studio 2022-ben
   - Indítsa el (F5)

A rendszer ezután használatra kész.
